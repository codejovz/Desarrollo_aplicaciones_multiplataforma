  10. ¿ Cuál de las siguientes afirmaciones sobre el almacenamiento de datos en Android utilizando SharedPreferences y SQLite es correcta?
    11. SharedPreferences es adecuado para almacenar grandes volúmenes de datos estructurados, mientras que SQLite se utiliza para datos de configuración y preferencias.
    SQLite permite el almacenamiento de datos de forma persistente y estructurada, mientras que SharedPreferences se utiliza para almacenar pares clave-valor de forma temporal.
    Los datos almacenados en SharedPreferences se pueden recuperar fácilmente con consultas SQL complejas, al igual que en SQLite.
    SharedPreferences y SQLite son métodos equivalentes para almacenar datos en Android, sin diferencias significativas en su uso.
