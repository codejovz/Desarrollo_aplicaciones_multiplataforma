ALGUNAS ILUSIONES ÓPTICAS

La imagen a la izquierda MarMon, tiene unas dimensiones de 50 x 50 pixels. ¿Reconoces el personaje?, es actriz y la palabra en negrita da una pista.
Sin embargo, al pulsar sobre la imagen se debe abrir otra página web para ver la imagen a tamaño original.
¿Puedes reconocer al personaje?, aunque la foto es la misma, parece ser que ya no se trata de una actriz sino de un famoso científico.

En la imagen de la derecha, aparece un tablero y sobre él un cilindro que da sombra. Una de las casillas a las que no afecta la sombra tiene escrita la letra A y otra de las casillas afectada por la sombra del cilindro tiene escrita la letra B.

¿Puedes decir si las dos casillas tienen el mismo color o diferente? Al pulsar sobre la imagen se debe abrir otra página web en la que aparecerán las casillas recortadas para comprobar si tienen el mismo color o no.
